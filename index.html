An exploratory collection of widgets

This is a small collection if simple web components (widgets) I created as I started to learn how to make them based on the documentation at Mozilla's MDN website. In each example I've attempt to explore a single aspect of the component. This project is hosted at GitHub at github.com/rsdoiel/widgets.
