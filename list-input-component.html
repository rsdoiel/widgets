A List Input Web Component Example

This is a demo of an "List component" that create an editable list from the JSON value passed in its "textContent" and the type of component to map the elements into indicated in its "type" attribute. The item component type should support setting/getting using a "value" attribute/property.

HTML Usage:

<list-input-component type="person-input-component"> <person-input-component given="R. S." family="Doiel"></person-input-component> <person-input-component given="Jane." family="Doe"></person-input-component> </list-input-component>

JavaScript Module list input component

Example JavaScript usage 

// Example using setter/getter "value"
let list_input = document.querySelector("list-input-component"),
    people = [ 
        { "given": "Jane", "family": "Doe" },
        { "given": "Ada", "family": "Lovelace" }
    ];
    
    // Update the list element to show two people
    list_input.value = people;

    // ... later "get" the list value and do something with.
    people = list_input.value;
